# Jagrat Ahuja – Developer Portfolio

This repository contains the source code for my personal developer portfolio website.

The website showcases my projects, skills, and achievements in computer science, software development, and educational technology. It acts as a central hub to present the systems and tools I build while exploring programming, problem-solving, and interactive software development.

**Live Website:**  
https://jagratahuja.vercel.app

---

## Tech Stack

The portfolio is built using modern web technologies:

- **Next.js** – React framework for building fast web applications  
- **Tailwind CSS** – Utility-first CSS framework for responsive and clean UI design  
- **Framer Motion** – Animation library used to create smooth interface interactions  

---

## Featured Projects

### StudyOS
A study efficiency analysis system that evaluates focus levels, study duration, and task completion to generate productivity insights for students.

### XSCAPE: The Equation Quest
An educational mathematics game designed to help students practice equations and inequalities through an interactive level-based gameplay system.

### Python Programs Collection
An archive of 100+ Python programs exploring algorithms, GUI applications, utility tools, and logic-based programming experiments.

## Running the Project

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

http://localhost:3000

## Deployment

The portfolio is deployed using Vercel.
